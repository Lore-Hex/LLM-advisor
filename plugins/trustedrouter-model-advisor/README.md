# TrustedRouter Model Advisor Plugin

This directory is the installable Claude Code and Codex plugin package. Both runtimes load the same advisor skill and connect to:

- TrustedRouter MCP for live catalog, provider, price, privacy, health, credit, and test-call data.
- AI IQ MCP for public model and benchmark data.

Set `TRUSTEDROUTER_API_KEY` before starting Claude Code or Codex. AI IQ does not require a key.

The canonical human-facing installation and usage documentation is in the [repository README](https://github.com/Lore-Hex/LLM-advisor#quick-start).
